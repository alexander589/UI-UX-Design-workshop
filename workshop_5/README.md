# Workshop_5

## Тема заняття
Криві Безьє, векторні об'єкти у Figma. Створення векторних значків (іконок).

## Хід роботи

### 1. Підготовка робочого середовища
Для початку роботи, у Figma я створив 5 фреймів розміром 24х24.

### 2. Створення іконок
Створив 5 іконок власноруч використовуя інструмент Pen для малювання кривих Безьє. Також де потрібно заокруглював кути за допомогою Corner Radius. Також за допомогою End points заокруглював кінці ліній.

На практиці я зробив:
