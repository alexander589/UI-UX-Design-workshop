# Workshop_7-8

## Тема заняття
Типографіка в дизайні інтерфейсів

## Хід роботи

### 1. Підготовка робочого середовища
Для початку роботи, у Figma я створив фрейм на базі інтерфейсу MacBook Air.

### 2. Опрацювання тексту
Я підібрав відповідні шрифтові стилі для заголовків, підзаголовків та основного тексту, після чого налаштував колірну гаму текстових елементів.

Також встановив оптимальні міжрядкові інтервали та відступи між абзацами, щоб забезпечити зручне сприйняття контенту.

Нумеровані списки оформлював вручну, щоб отримати форматування 01, 02, 03, яке відрізняється від стандартної нумерації.

Для виділення автора застосував підкреслення до тексту з його ім’ям.

Також за допомогою булевих операцій я створив іконку чоловіка та елемент індикації прогресу, щоб доповнити загальний вигляд макета. Додав лапки та зробив їх відповідного кольору, та додав з іконок стрілку.
